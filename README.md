## Features

| Feature | Description |
|---|---|
| 🗂️ Full Bracket | Round of 32 → Final (31 matches, 32 teams) |
| 🖱️ Click to Advance | Click any team to send them to the next round |
| 🔁 Round Unlock | Next round unlocks automatically after completing current |
| 🌍 Bilingual | Full Arabic & English UI with RTL support |
| 🥇 Awards Section | Pick Top Scorer, Best Player, Best Goalkeeper |
| 📸 PNG Export | Download your full bracket as a high-res image |
| 🎉 Confetti | Champion celebration animation on final pick |
| 📱 Mobile Friendly | Tab-based navigation on small screens |
| ✨ Particle BG | Animated gold particle background |
| 0️⃣ Zero Dependencies | Pure HTML/CSS/JS — no frameworks, no npm |

## Live Demo

**[🏆 Open Bracket →](https://AyhamMegdadi.github.io/worldcup2026)**

## How to Use

1. Click any team in the **Round of 32** to advance them
2. Complete the round → next round **unlocks automatically**
3. Continue until you pick your **World Cup Champion** 🏆
4. Select **Top Scorer, Best Player, Best Goalkeeper**
5. Click **Download** to save your prediction as PNG

## Run Locally

```bash
git clone https://github.com/Ayham-Megdadi/worldcup2026.git
cd worldcup2026
# Just open index.html in your browser — no server needed
open index.html
```
